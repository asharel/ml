Introducción¶

En esta prctica se introduce una serie de ejercicios para estudiar las tcnicas de reduccin de dimensionalidad. Seguiremos trabajando con los datos de la prctica anterior realizando un estudio para que intentemos determinar que parmetros son los mejores para un ejercicio de clasificacin.

Objetivo¶

El objetivo de esta prctica es que adquiramos conocimientos bsicos para solventar problemas de reduccin de dimensionalidad de los datos. Esto se enmarca dentro del rea de preparacin de datos previos a un algoritmo de machine learning.

Trabajo Desarrollado¶

Tras una preparacin previa de los datos que realizamos segn lo aprendido en la prctica anterior, analizamos distintos mtodos de reduccin de dimensionalidad con el objetivo de aplicar luego un algoritmo de aprendizaje y evaluar la eficacia de dicho algoritmo en funcin de las variables eliminadas segn los mtodos descritos. As somos capaces de evaluar los mtodos aplicados de reduccin de dimensionalidad.

def specificity(y_true, y_pred):
    tn, fp, fn, tp = metrics.confusion_matrix(y_true, y_pred).ravel()
    return tn/(tn+fp)
